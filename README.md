# frontend-vuejs

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Obs.: The server must run in 8081 port due CORS API rules.

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
